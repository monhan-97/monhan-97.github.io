# 绘制

在浏览器中，绘制(Painte)是指将网页的视觉内容渲染到屏幕上的过程

## 重绘

重绘通常发生在以下情况下：

- **样式更改**:

  - 当通过 `JavaScript` 或者 `CSS` 修改了元素的样式，但是**没有改变其几何属性（比如位置、大小）**，浏览器会触发重绘。
  - 例如，修改了`背景颜色`、`文字颜色`、`边框`等属性。

- **伪类激活**

  - 激活伪类`(如:hover)`或者伪元素`(如::before、::after)`时，也会触发元素的重绘。
  - 例如，修改了`背景颜色`、`文字颜色`、`边框`等属性。

重绘的过程并不会重新计算元素的大小和位置（这称为重排或回流）。

因此，重绘的性能开销通常比重排小，但在频繁触发时仍可能影响页面的响应速度和流畅度。
