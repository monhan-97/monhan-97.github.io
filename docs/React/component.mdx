# 组件和组件属性

组件: 包含内容，样式和功能的 UI 单元

## 创建一个单元

:::warning
组件的名称首字母必须大写
:::

1. 函数组件

返回一个 React 元素

2. 类组件

必须继承 React.Component
必须提供 render 函数 用于渲染组件

## 组件的属性

1、 对于函数组件，属性会作为一个对象的属性 传递给函数的参数
2、 对于类组件属性会作为一个对象 传给类的构造函数的参数

注意:组件的写法必须要使用小驼峰命名法

:::warning
组件无法改变的自身的属性 数据属于谁 谁才有权利改动
:::

:::warning
React 的数据 自顶而下的流动
:::
