# JSX

## 什么是 JSX

- FaceBook 起草的扩展语法
- 本质是一个 JS 对象, 会被 Babel 编译。
  - 在 React17 之前 在最终会被转换成 React.createElement
  - 在 React17 之后 没有手动导入 React 会被使用 react/jsx-runtime 进行转换
- 每个 JSX 表达式，有且仅有一个根节点。
  - React.Fragment
- 每个 JSX 元素必须结束(XML 规范)

## 在 JSX 嵌入表达式

- 在 JSX 使用注释
- 将表达式作为内容的一部分
  - null、undefined、false 不会显示
  - 普通对象、不可以作为子元素
  - 可以放置 React 元素对象
- 将表达式作为元素属性
- 属性使用小驼峰命名法
- 防止注入攻击
  - 自动编码
  - dangerouslySetInnerHTML

## 元素的不可变性

- 虽然 JSX 是一个对象，但是对象中的所有属性不可更改
- 如果确实需要更改元素的属性，需要重新创建 JSX 元素
