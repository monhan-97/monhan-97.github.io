# Null 类型

Null 类型 只有一个值 就是特殊值 `null`,从逻辑上讲 `null` 值表示一个空对象指针，这也是给 `typeof`传一个 `null` 发返回 `object` 的原因

```javascript
let message = null;
console.log(type message); //object;
```

在定义将来要保存对象值的变量的时候，建议使用 `null` 来初始化，不要使用其他值。
这样只要检查这个变量的值是不是 `null`, 就可以知道这个变量是否在后来被重新赋值

`undefined`是由 `null` 派生而来的,因此它们表面相等

```javascript
console.log(null == undefined); //true;
```
