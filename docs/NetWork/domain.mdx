# 域名

## 什么是域名

**域名**是一种人类可读的地址,用于在互联网上标识和访问网站或其他网络资源。

**域名**使用户能够通过易记的名称`(如 www.example.com)`访问计算机，而不是通过数字形式的 IP 地址`(如 192.0.2.1)`。

## 域名的类型

**域名**的类型有以下几种:

- **根域名**:`.`
- **顶级域**:最右边的部分 如`.com`、`.org`、`.net`、`.edu`、`.cn` 等
- **二级域名**:紧接在顶级域之前的部分，通常用于标识特定组织或企业 如`com.cn`,`baidu.com`,`jd.com`
- **子域**:在二级域之前的部分，用于进一步划分域名，帮助组织结构化内容或服务,如`blog.example.com`

## 域名系统

**DNS(Domain Name System)** 是将域名转换为 **IP** 地址的系统。用户输入域名 **DNS** 会将其转换为计算机可以识别的 **IP** 地址,从而访问相应的服务器。
